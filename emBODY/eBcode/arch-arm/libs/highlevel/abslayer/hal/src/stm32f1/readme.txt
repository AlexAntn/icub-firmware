In this folder there are:
 - root: 
   all source files that implement hal functionalities.
 - folder stm32_utility: 
   contains stm32 file that aren't included into stm32.lib.
   They have been modified where is the tag IIT_EXT.
 - folder MCBSTM32C_board:
   contains definition for that specific board.
 - folder EMS001_board: 
   contains definition for that specific board.
 - folde reference:
   contains the original files used to implement some hal_* drivers.
